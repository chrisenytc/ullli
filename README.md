## Getting Started

1º Clone ullli repo

```bash
$ git clone git@github.com:chrisenytc/ullli.git
```

2º Enter in ullli directory
```bash
$ cd ullli
```

3º Configure the settings in the env files

4º Run setup

```bash
$ npm run setup
```

5º Run ullli

```bash
$ docker-compose up server
```

Test your ullli app

```bash
$ docker-compose up test
```
